    ## Fetch a Bot

    ```sh
    GET api.topstats.gg/bots/:id
    ```

    #### Query Parameters
    ```json
    {
        "id": "<string>"
    }
    ```

    ## Fetch a Bot's Historical Data (Graphs)

    ```sh
    GET api.topstats.gg/bots/:id/historical
    ```

    #### Query
    ```json
    {
        "id": "<string>"
    }
    ```

    #### Parameters
    ```json
    {
        "timeFrame": "<string>" ['alltime','5y','3y','1y','90d','30d','7d','3d','1d','12h','6h']
        "type": "<string>" ['monthly_votes','total_votes','server_count','shard_count']
    }
    ```
