  ## Fetch rankings for the top bots

  ```sh
  GET api.topstats.gg/discord/rankings/bots
  ```

  #### Parameters
  ```json
  {
    "limit": "<number>" default: 100 | max: 500,
    "sortBy": "<string>" [
      'monthly_votes_rank',
      'total_votes_rank',
      'server_count_rank',
      'shard_count_rank'
    ],
    "sortMethod": "<string>" ['asc', 'desc'],
    "offset": "<number>"
  }
  ```
