    ## Fetch a Widget

    ```sh
    GET https://widgets.topstats.gg/custom/:type
    ```

    #### Query Parameters
    ```json
    {
      "name": "<string>",
      "type": "<string>" [
        'graph',
        'graph.png,
      ],
      "name": "<string>",
      "data_points": "<JSON:string>",
      "color": "<string>"
    }
  ```

  ## Usage Example

    ```sh
    https://widgets.topstats.gg/custom/graph.png?name=<string>&data_points=<JSON>
    ```
