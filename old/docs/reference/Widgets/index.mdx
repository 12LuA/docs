    ## Fetch a Widget

    ```sh
    GET https://widgets.topstats.gg/:id/:statistic?timeframe=:timeframe
    ```

    #### Query Parameters
    ```json
    {
      "id": "<string>",
      "statistic": "<string>" [
        'servers',
        'monthlyvotes',
        'totalvotes',
        'servers.svg',
        'monthlyvotes.svg',
        'totalvotes.svg',
        'servers.png',
        'monthlyvotes.png',
        'totalvotes.png'
      ],
      "timeframe": "<string>" [
      'alltime',
        '5y',
        '3y',
        '1y',
        '270d',
        '180d',
        '90d',
        '30d',
        '7d',
        '3d',
        '1d',
        '12h',
        '6h'
      ],
    }
  ```
