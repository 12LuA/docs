All requests to the API require you to provide an API key which you can find on your when you login to [TopStats.gg](https://topstats.gg).

Once you have logged in, click on your profile picture at the top right, and click "Get Token". Agree to our terms and click copy token!


## How To Send My Token
All of your requests to our API must contain a valid API key and provided through the `Authorization` header.

```json
{
  "Authorization": "My Super Secret Key"
}
```

## My Token Has Been Stolen, What Do I Do?
You can reset your token by going to the Get Token Popup and Click the reset token button in the token actions section!
